# Fruit Plant Disease Detection System

A web-based machine learning project for detecting fruit plant leaf diseases using React, Django REST Framework, and TensorFlow/Keras.

## Developed By

Devakrupa Parashuramsingh Rajput  
UUCMS No: P02DR2412602

## Project Features

- Upload leaf image from gallery
- Capture image using camera
- Detect fruit plant disease
- Display prediction result
- Show disease recommendation
- React frontend
- Django REST API backend
- TensorFlow/Keras ML model integration

## Technology Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Django
- Django REST Framework
- Python

### Machine Learning
- TensorFlow/Keras
- EfficientNetB0 / MobileNetV2
- Image Classification
